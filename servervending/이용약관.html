Server Vending 이용 약관

이용약관 업데이트 기록: [클릭]

제 1조 (목적)

본 이용약관은 Server Vending 구매자 간에 Server Vending 이 제공하는 서비스의 가입 및 이용에 권한 권리, 의무, 책임 사항을 규정하기 위한 것입니다.

제 2조 (용어의 정의)

1. 디스코드 (discord): discord.com 에서 제공하는 온라인 메신저
2. 구매자 (Buyer): Server Vending 서비스를 구매하고, 사용하는 사람
3. 연장: 만료일을 연장하는 것
4. 계좌 자충: 회원이 지정한 계좌에 입금이 되면 이체를 한 자에게 자판기 잔액을 충전 해주는 서비스
5. RCON (RCON): 구매자 가 이용하는 게임 서버에 연결 해주는 서비스

제 3조 (이용 자격)

1. 운영자가 허가한 구매자에 한 해 사용료를 운영자에게 한달 마다 납부 하면 사용할 수 있습니다.

제 4조 (서비스 가입)

1. 서비스 이용을 희망 하는 이용자는 운영자를 통해 가입할 수 있으며, 주체의 행위의해서만 유효하며, 구매 시 약관에 동의하는 것으로 간주됩니다.
2. 서비스 가입시 다음과 같은 정보를 수집할 수 있습니다.
 2.1 디스코드 서버 정보 (고유번호, 닉네임, 등...)
 2.2 계좌번호
 2.3 서버 아이피
 2.4 서버 포트
 2.5 구매자 디스코드 아이 정보 (고유번호, 닉네임)

제 5조 (서비스 가입 거절)

1. 불법 서버를 운영 하는 경우
2. 기타의 이유로 이용자가 서비스 이용에 적합 하지 않다고 운영자가 판단한 경우

제 5조 (개인 정보의 변경, 보호)

1. 가입 신청자가 또는 구매자는 이용 신청 시 기재한 정보가 변경되었을 경우에는 즉시 운영자에게 수정 요청해야 합니다.
2. 운영자는 구매자의 개인 정보를 서비스 제공을 위해서만 수집하며, 법령에서 정보 공개를 요구 하거나 가능한 경우에만 정보를 제공 할 수 있습니다.

제 7조 (서비스의 내용)

1. Server Vending 은 구매자에게 다음과 같은 서비스를 제공 합니다.
 1.1 디스코드 내 자동 판매 및 자동 충전
 1.2 자동 판매한 상품을 RCON 으로 보내는 작업

제 8조 (서비스 이용 시간)

1. 서비스는 업무상 또는 기술상 장애, 기타 특별한 사요 없는 한 연중 무휴, 24시간 이용 할 수 있습니다.
2. 제공하는 서비스 중 일부에 대한 이용시간을 별도로 정할 수 있으며, 이 경우 관련사항을 사전에 구매자에게 공지 또는 통지 합니다.
3. 구매자 한에 개인이 봇을 운영하는 경우 이용 시간 제한이 없습니다.

제 9조 (서비스 제공의 중지)

1. 다음과 같은 상황 일 때 서비스 제공을 중지 할 수 있습니다.
 ● 서비스용 설비의 보수 등 공사로 인한 부득이한 경우
 ● 국가 비상상태
 ● 정전
 ● 서비스 설비의 장애 또는 서비스 이용의 폭주 등으로 정상적인 서비스 이용에 지장이 있는 경우
 ● 전기 통신 사업 법에 규정된 기간 통신 사업자가 서비스를 중지했을 경우
2. 모든 서비스 구매자는 Server Vending 의 정책에 따라 변경 및 중지가 가능하며, 또한 이로 인해 발생하는 문제에 대하여 Server Vending 은 어떠한 책임도 지지 않습니다.

제 10조 (서비스의 불법 사용 금지)

1. 구매자는 서비스 이용 권한 및 이용 환경을 타인에게 양도 및 증여 할수 없으며, 또한 이를 어떠한 조건의 담보로도 제공할 수 없습니다.
2. 구매자는 관계 법령 및 본 약관의 내용과 Server Vending 이 공지하는 사항을 준수 해야합니다.
3. 구매자는 Server Vending 의 사전 승낙 없이 서비스를 이용하여 어떠한 영리 행위도 할수 없으며, Server Vending 의 서비스를 이용하여 얻은 정보를 수정, 복사, 번역,충판,실행,양도, 배포, 판매, 및 기타의 방법으로 사용하거나 타인에게 제공할수 없습니다.
